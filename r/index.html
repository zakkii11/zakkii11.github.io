Handling Japanese Calender

{wareki}

{wareki} helps you working with documents which include Japanese calender. The packages will be useful when you want to replace Japanese calender in character data into Western calender. It would also useful for modifying orthographical variants in the documents.

Installation
You can install {wareki} from my GitHub Repo.
 library(devtools)
 devtools::install_github("zakkii11/wareki")
 library(wareki)
Vignette
 wareki("恐らく平成20年生まれの人は、令和元年にはもう11歳になっているのだろう。")
[1] "恐らく2008年生まれの人は、2019年にはもう11歳になっているのだろう。"
